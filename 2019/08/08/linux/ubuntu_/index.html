    ubuntu18.04server给eth0增加ip地址

    参考：https://netplan.io/

    cd /etc/netplan/
    在这个目录下有网卡相关的yaml配置文件
    cat /etc/netplan/99-eth0.yaml
    直接添加一行 addresses: [x.x.x.x/x]
    即可配置一个网卡新增一个IP
